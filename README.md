##  About the Project

Modern and responsive landing page developed with React, featuring smooth animations, glassmorphism design, and an exceptional user experience.

##  Features

-  Modern and responsive design
-  Smooth CSS animations
-  Glass morphing effects
-  Smooth scrolling navigation
-  Optimized performance
-  Mobile-first approach

##  Technologies Used

- **React 18** - JavaScript library for interfaces
- **Babel Standalone** - JSX transpilation in the browser
- **CSS3** - Modern animations and layouts
- **HTML5** - Semantic framework

##  How to run

1. Clone the repository:
```bash
git clone https://github.com/JoguisSan/landing-page-react.git
```
2. Navigate to the directory:
```bash
```cd landing page-react
```
3. Open the `index.html` file in your browser or use a local server:

```bash
# With Python 3
python -m http.server 8000

# With Node.js
npx serve

```

4. Access `http://localhost:8000` in your browser

##  Responsiveness

The project is fully responsive and optimized for:

-  Mobile (320px - 768px)
-  Tablet (768px - 1024px)
-  Desktop (1024px+)
